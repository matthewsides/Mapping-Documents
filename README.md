# Mapping-Documents

This area holds mapping documents that cover specific set criteria, offering links to evidence that has been deemed to meet that specific criteria, with said evidence being linked to GitHub README.md documentation, linkedin articles, external files stored in GitHub repositorys, etc.

#### Download:
The mapping documents are stored on GitHub servers and may be download through clicking the above links, all the documents are word documents and may open automatically if set to do so otherwise the documents should be download and moved to the computers pre-set downloads file on your hard drive.
